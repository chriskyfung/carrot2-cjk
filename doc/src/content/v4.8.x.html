Carrot2 4.8.x

Release history for Carrot2 4.8.0 and bugfix releases.

Version 4.8.5

The 4.8.5 release upgrades Jetty to 12.x, Jackson to 2.21.1 and other dependency and build infrastructure updates. A new option to start the DCS with just the service has been added.

Jetty version strings omitted

The DCS will no longer return jetty version strings in error pages or HTTP headers.

New --service-only option for the DCS

The DCS launcher accepts a new --service-only option that starts the server with only the /service context, omitting the demo web application and other bundled contexts.

Version 4.8.4

The 4.8.4 release includes an update of dependencies, build plugins and gradle build system.

Version 4.8.3

The 4.8.3 release fixes the user interface to correctly pass the user-provided partner ID to the eTools search service.

This is an internal, non-maven release.

Version 4.8.2

The 4.8.2 release fixes the user interface to correctly pass the user-provided customer ID to the eTools search service.

This is an internal, non-maven release.

Version 4.8.0

This release includes an upgrade of dependencies, build plugins and a switch to Java 21 as the minimum runtime platform (a consequence of switching to Lucene 10).

Dependency upgrades

jackson to 2.19.0, jetty to 11.0.25, Lucene to 10.2.1, slf4j to 2.0.17, log4j to 2.24.3 (and other transitive updates).
